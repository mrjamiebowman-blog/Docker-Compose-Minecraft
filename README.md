# Docker Compose: Minecraft (Windows)
This is a docker compose script to run minecraft. 

## Requirements
* Docker Desktop
* Visual Studio Code
* PowerShell
* GitHub Account

## GitHub: SSH Keys
Generate an ssh key to use in GitHub.
`$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"`   

`type ~/.ssh/id_rsa.pub`

start.ps1
